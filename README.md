# Makdownアプリです
Markdown形式のデータを管理します。
Typescript練習用に作成しました。

## 環境
環境は以下を使っています。

* React
* Flask
* MariaDb

## 詳細について
以下のブログに情報を載せています。
[40代から始めるブログ](https://bsf40.blogspot.com/)

# 起動方法
```
# docker-compose up
```
