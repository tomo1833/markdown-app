# Makdownアプリです
Markdown形式のデータを管理します。
Typescript練習用に作成しました。

## 環境
環境は以下を使っています。

* React
* Flask
* MariaDb

## 詳細について
以下のブログに情報を載せています。
[40代から始めるブログ](https://bsf40.blogspot.com/)


## cloneで実行する方法（検討中）
```
git clone git@github.com:tomo1833/markdown-app.git

cd markdown-app

docker-compose run --rm front sh -c "cd markdown-app && npm i"

docker-compose up
```
